# 📱 Android Projects Monorepo

This repository contains multiple Android applications developed using Android Studio.  
Each sub-directory represents a complete and independent Android project that can be opened and run individually.

---

## 📂 Repository Structure

Android-Projects/
├── BUS/
├── Certificate/
├── DrawingShapes/
├── MovieBooking/
├── Notification/
├── TextEditor/
└── README.md

Each folder contains its own app module, Gradle files, and configuration.

---

## 🚀 Getting Started

Follow the steps below to download and run any application from this repository.

---

## 🛠 Requirements

- Android Studio (latest stable version recommended)
- Java JDK 8 or higher
- Android SDK
- Gradle (included with Android Studio)
- Android Emulator or Physical Device

---

## ⬇️ Download the Repository

### Option 1: Clone using Git
git clone https://github.com/YOUR-USERNAME/Android-Projects.git


### Option 2: Download ZIP
1. Click on **Code**
2. Select **Download ZIP**
3. Extract the ZIP file

---

## ▶️ How to Run an App

1. Open Android Studio
2. Click **File → Open**
3. Navigate to: Android-Projects/<App-Name>
Example:Android-Projects/BUS
4. Click **OK**
5. Wait for Gradle sync to complete
6. Connect an emulator or physical device
7. Click **Run ▶**

---

## 🧩 Apps Included

| App Name | Description |
|--------|-------------|
| **BUS** | Android application to view bus timings, display the top two upcoming buses based on the current time, and provide notification alerts for bus schedules. |
| **Certificate** | Application to generate and manage course completion certificates digitally. |
| **DrawingShapes** | Interactive app that allows users to draw and visualize different geometric shapes. |
| **MovieBooking** | Application used to store and manage movie booking details. |
| **Notification** | Sample Android application demonstrating notification creation and delivery. |
| **TextEditor** | Text editor app that supports basic text operations such as editing, formatting, and manipulation. |


## ⚠️ Important Notes

- Each app is independent and has its own Gradle configuration
- Do not open the root folder directly to run apps
- local.properties and build folders are excluded from version control
- API keys (if required) must be configured manually

---

## 🧪 Troubleshooting

- Gradle sync failed → Update Gradle and Android plugin
- SDK not found → Configure SDK path in Android Studio
- Build error → Invalidate cache and restart Android Studio
